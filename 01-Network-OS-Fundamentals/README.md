# Network and OS Fundamentals (Security Engineer)

This folder covers core networking and operating system concepts from a security engineering perspective.
The focus is on how network and OS artifacts are used during detection, investigation, and incident response.

## What this section supports
- SIEM alert investigation and validation
- Log analysis and enrichment
- Incident response decision-making
- Mapping telemetry to attacker behavior

## Contents
- Networking for security investigations
- Network troubleshooting during incidents
- Authentication and access patterns
- Linux permissions, processes, and logs
- Windows authentication and event analysis
